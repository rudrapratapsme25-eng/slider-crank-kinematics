# Slider-Crank Mechanism Kinematics ⚙️

This repository contains the programmatic simulation for a single-cylinder slider-crank mechanism, developed as part of the Experiential Learning curriculum for the  RV College of Engineering. 

### Project Overview
The Python script extracts geometric constraints from a fully constrained 3D CAD model to visualize the conversion of uniform rotary motion into linear reciprocating motion. 

**Key Features:**
* Computes real-time position kinematics using trigonometric constraints.
* Visualizes the continuous 1-Degree-of-Freedom motion cycle using `Matplotlib`.
* Analyzes the non-sinusoidal displacement of the piston caused by connecting rod angularity.

**Tech Stack:** Python, NumPy, Matplotlib
